# 🧑‍💻 Interactive Resume Website (Hosted on AWS)

A modern HTML + TailwindCSS resume website hosted securely using AWS services with HTTPS and a custom domain.

---

## 🌐 Hosting Stack

- Amazon S3 (static hosting)
- Amazon CloudFront (CDN)
- Route 53 (DNS management)
- AWS Certificate Manager (SSL/TLS)
- GitHub (source control)

---

## 🚀 Features

- Clean, responsive resume page using HTML + TailwindCSS
- Custom domain `https://tinyurl.com/4c4wtt6d`
- Deployed with HTTPS via CloudFront + S3
- Global CDN with caching + fast load times

---

## 🔧 Setup Steps

1. Upload your site files to S3 bucket
2. Enable static website hosting
3. Create CloudFront distribution with S3 as origin
4. Use Route 53 to map your custom domain
5. Use ACM to issue SSL certificate and attach it to CloudFront
6. Deploy changes via Git or AWS Console
